❌ "Error: Could not create repository"
   → Probablemente el nombre "autoradar-dashboard" ya existe
   → Intenta: autoradar-dashboard-v2, autoradar-app, etc
   → O borra el repo anterior y crea de nuevo

❌ "Archivos no aparecen en GitHub"
   → Espera 5 segundos y recarga la página (F5)
   → Si persiste, intenta subir de nuevo desde "Add file" → "Upload files"

❌ "No puedo seleccionar 13 archivos a la vez"
   → Sube en 2 tandas:
     • Primero los 7 principales (index.html, car-tracker, config, etc)
     • Luego la documentación (.md, .txt)

❌ ".gitignore no aparece"
   → Algunos sistemas ocultan archivos que empiezan con punto
   → Ve a GitHub → Settings → confirma que está ahí
   → O súbelo nuevamente como "gitignore" (sin punto)
